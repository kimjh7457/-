# ⏰ 알람 앱 - 함께 해제해야 꺼지는 재미있는 알람

재미있는 기능이 있는 알람 앱입니다! 알람이 울리면 여러 번 클릭해야 해제되는 재미있는 시스템을 가지고 있습니다.

## 🚀 주요 기능

- **현재 시간 표시**: 실시간으로 현재 시간을 보여줍니다
- **알람 설정**: 원하는 시간에 알람을 설정할 수 있습니다
- **알람 관리**: 알람을 활성화/비활성화하고 삭제할 수 있습니다
- **재미있는 해제 시스템**: 알람이 울리면 2-4번 랜덤하게 클릭해야 해제됩니다
- **반응형 디자인**: 모바일과 데스크톱에서 모두 사용 가능합니다

## 🎮 사용법

1. **알람 추가**: 시간을 선택하고 "알람 추가" 버튼을 클릭합니다
2. **알람 관리**: 각 알람의 시작/중지 버튼으로 알람을 제어할 수 있습니다
3. **알람 해제**: 알람이 울리면 모달이 나타나고, 지정된 횟수만큼 클릭해야 해제됩니다

## 🛠️ 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 미리보기
npm run preview
```

## 🎨 기술 스택

- **React 18** - 사용자 인터페이스
- **TypeScript** - 타입 안전성
- **Vite** - 빠른 개발 환경
- **Lucide React** - 아이콘
- **CSS3** - 스타일링

## 🎯 특별한 기능

### 함께 해제 시스템
알람이 울리면 랜덤하게 2-4번의 클릭이 필요합니다. 이는 사용자가 정말로 일어나야 한다는 의지를 보여주는 재미있는 기능입니다!

### 시각적 피드백
- 알람이 울릴 때 화면이 펄스 애니메이션으로 깜빡입니다
- 진행 상황을 실시간으로 확인할 수 있습니다
- 모던하고 아름다운 UI/UX

## 📱 반응형 지원

모바일과 데스크톱 모두에서 최적화된 경험을 제공합니다.

## 🎵 알람 소리

내장된 알람 소리가 포함되어 있어 별도의 파일 없이도 알람 기능을 사용할 수 있습니다.
